# 🎲 Pig Game

A fun and interactive two-player dice game built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. Players take turns rolling a dice, accumulating points until they choose to hold or risk losing their current score by rolling a **1**. The first player to reach **100 points** wins the game.

## ✨ Features

- 🎲 Roll a virtual dice
- 👥 Two-player gameplay
- ➕ Accumulate current score
- 💾 Hold score to save progress
- ❌ Rolling a **1** resets the current score and switches turns
- 🏆 First player to reach **100 points** wins
- 🔄 Start a new game anytime
- 📱 Responsive design

## 🎮 How to Play

1. Player 1 starts the game.
2. Click **Roll Dice** to roll.
3. If the dice shows **2–6**, the value is added to the current score.
4. If the dice shows **1**, the current score is lost and the turn switches to the other player.
5. Click **Hold** to add the current score to the total score and pass the turn.
6. The first player to reach **100 points** wins!
